# VolatilityForecasting
This package has been created to forecast currency volatility using a variety of models:

1. Shifting 1 element
2. Linear regression
3. Arch
4. Garch
5. VAR
6. KNN
7. LASSO (in R)
8. Ordered LASSO (in R)
